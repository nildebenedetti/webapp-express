# iScream - La Gelateria da Urlo

## Brief Assegnato

## iScream 😱 👻
iScream è una gelateria che vive sul doppio significato tra “ice cream” e “I scream”, quindi perfetta per una comunicazione pop, colorata e un po' drammatica a tinte horror.

Si possono proporre prodotti come:

Pistacchio dell'Apocalisse;
Fragola Urlante;
Triple Chocolate Breakdown;
Limone Esistenziale.

Categorie possibili:

Coni
Coppette
Granite
Gusti speciali
Limited edition
